# Fuzzy C-Means Clustering on Butterfly Dataset

This repository implements the Fuzzy C-Means (FCM) clustering algorithm on a small butterfly-shaped dataset.

The project demonstrates:

- Fuzzy clustering concepts
- Membership matrix computation
- Iterative center updates
- Euclidean distance calculations
- Visualization of fuzzy cluster assignments

Unlike hard clustering methods, FCM allows each point to belong to multiple clusters with different membership degrees.

---

# Features

- Two-cluster fuzzy partitioning
- Membership matrix updates
- Cluster center optimization
- Butterfly-shaped synthetic dataset
- Visualization using color mapping

---

# Technologies

- Python
- NumPy
- Matplotlib

The algorithm generates:
